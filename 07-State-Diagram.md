**状态转换表:**

| 当前状态 | 触发事件 | 目标状态 | 附加操作 |
|----------|----------|----------|----------|
| 注册 | 注册成功 | 未认证 | 创建账户、发送欢迎消息 |
| 未认证 | 实名认证通过 | 正常 | 解锁完整功能、补发推荐 |
| 正常 | 购买VIP | VIP | 即时开通、推送VIP权益 |
| VIP | VIP过期 | 正常 | 保留历史记录、提示续费 |
| 正常/VIP | 异常行为检测 | 冻结 | 限制匹配/聊天、通知用户 |
| 冻结 | 申诉通过 | 正常 | 恢复功能、清除警告 |
| 正常/VIP/冻结 | 严重违规 | 封禁 | 永久封禁、清理推荐列表 |
| 正常/VIP | 主动注销 | 已注销 | 30天冷静期、数据暂存 |
| 已注销 | 30天内重新登录 | 正常 | 恢复账号和数据 |

**匹配操作状态说明:**

| 状态 | 描述 | 可执行操作 |
|------|------|------------|
| Ready | 推荐已生成待浏览 | 打开App查看 |
| Showing | 正在展示某用户卡片 | 滑动心仪/跳过、查看详情 |
| Skip | 已跳过该用户 | 5秒内可撤销 |
| Liked (Pending) | 已心仪，等待对方 | 无（等待对方操作） |
| Matched | 双向心仪成功 | 开始聊天、查看对方资料 |
| Chatting | 聊天中 | 发消息、AI教练、视频通话 |
| Dating | 线下约见中 | 约会提醒、互评 |
| Unmatch | 解除匹配 | 删除聊天记录、重新匹配 |
